Sobre Este Proyecto

Objetivo del Sitio

Este sitio web demuestra el uso de etiquetas semánticas HTML5 y aplicación de estilos con CSS. El proyecto incluye diversos elementos multimedia, listas, tablas y formularios.

Aprenderemos a estructurar correctamente un documento web profesional y a publicarlo en GitHub Pages.

Tecnologías Utilizadas

Lista Ordenada de Pasos:

  1. Configuración del entorno de desarrollo
  2. Creación de estructura HTML5
  3. Aplicación de estilos CSS
  4. Publicación en GitHub Pages

Herramientas Necesarias:

  • Visual Studio Code
  • Navegador Web Moderno
  • Git y GitHub
  • Live Server Extension
